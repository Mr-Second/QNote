# QNote v0.1.0

首个公开 Release — MVP 阶段完成。

## 下载

| 文件 | 说明 | 体积 |
|---|---|---|
| `QNote-Setup-0.1.0-x64.exe` | NSIS 安装包（推荐） | ~17 MB |
| `QNote-Portable-0.1.0-x64.zip` | Portable 绿色版（解压即用） | ~23 MB |

## 系统要求

- Windows 10/11 x64
- 无需预装运行时

## 新功能（首版）

### 核心
- 便签创建 / 编辑 / 删除（含确认弹窗）
- 富文本编辑器：加粗 / 斜体 / 下划线 / 删除线 / 列表 / 对齐 / 字体 / 字号 / 颜色
- 图片插入：本地文件 + PicGo + 剪贴板粘贴
- 全文搜索（Xapian 中文分词 + 高亮）
- 分类管理：自定义图标 / 改名 / 删除 / 移动
- 数据备份/恢复（7z + 可选 AES-256 加密）

### 系统集成
- 系统托盘（显隐 / 右键菜单）
- 窗口贴边隐藏
- 开机自启动
- 响应式布局（三栏自适应缩放）

### 视觉
- 纸张质感背景（QML Canvas）
- 异形窗口（自定义标题栏 + SVG 角标）
- Light / Dark / System 三态主题切换

### 稳定性
- 按天滚动日志（`%APPDATA%\QNote\QNote\logs\`）
- 崩溃捕获（SEH + CRT + 未捕获异常 → minidump）
- 启动自检（版本/环境/Manager 加载状态）

### 其他
- 多语言：简体中文 / English，运行时切换
- 安装包：UAC 提权 + 运行检测 + 双语 UI + 卸载询问删用户数据

## 已知限制

- 仅 Windows x64
- 未代码签名（SmartScreen 可能警告）
- 无自动更新检查（需手动下载新版本）
- 单实例未强制（多实例会导致托盘重复 + SQLite 锁）

## 升级

首版，无升级流程。后续版本安装版直接覆盖安装即可（用户数据保留）。
