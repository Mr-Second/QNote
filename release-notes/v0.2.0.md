# QNote v0.2.0 Release Notes

## 🎉 新增功能

### 贴边隐藏时隐藏任务栏图标
- 设置页新增「贴边隐藏时隐藏任务栏图标」开关（默认关闭）
- 开启后，窗口贴边隐藏时 Windows 任务栏图标自动消失
- 窗口显现后任务栏图标自动恢复
- 开关修改立即生效，无需等待下次贴边隐藏周期
- 设置持久化保存

## 🔧 技术改进

### Windows 任务栏图标控制
- 使用 Windows **ITaskbarList3 COM 接口**（DeleteTab/AddTab）控制任务栏图标
- 不改变窗口类型或样式，完全不影响窗口行为
- 不破坏热区检测、鼠标事件、焦点处理
- 新增 `WindowsTaskbarHelper` 平台辅助类封装 Windows API

### 版本号从 git tag 自动派生
- 版本号单一真相源改为 git tag，`xmake.lua` 用 `git describe --tags --abbrev=0` 运行时派生
- 消除手动维护版本字段导致的漏改风险（修复了本次 v0.2.0 安装包 DisplayVersion 仍为 0.1.0 的问题）
- release CI 新增 `Verify version resolves from tag` 步骤，确保 tag 与构建版本一致
- 未来发版只需打 tag，无需修改任何源文件
